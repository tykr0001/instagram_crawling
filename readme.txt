입력에 태그를 입력(쉼표로 구분하고 띄어쓰기는 모두 무시함)
ex) 분식, 맛나분식, 덕천


태그가 모두 포함된 게시글을 검색합니다
검색은 가장 먼저 작성한(제일 왼쪽) 태그를 기준으로 합니다
따라서 속도를 빠르게 하고 싶다면 검색 결과가 제일 적은 태그를 가장 앞에 적는 것이 좋음


chromedriver는 해당 환경의 크롬 버전과 동일한 버전을 다운
(폴더에 포함돼있는 버전은 85)
다운로드 링크:https://chromedriver.chromium.org/downloads
